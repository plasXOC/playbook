## 🖥️ What is PLAS?

PLAS is a custom AME Wizard playbook that strips Windows down to what actually matters. Telemetry, background garbage, and unnecessary components are removed at the source — not just toggled off until the next update undoes everything.

Built by people who know what a properly tuned system feels like. Not benchmarks. Not placebo tweaks. Real, tangible differences you can feel the moment you sit down.

---

## ⚡ Why PLAS?

### 🔒 Privacy First
Telemetry and data collection aren't just disabled — they're gone. Microsoft doesn't get to phone home on your machine.

### 🚀 Real Performance
Every change in this playbook exists for a reason. No fluff, no placebo registry tweaks, no snake oil. Just a leaner, faster Windows that stays out of your way.

### 🎛️ You're in Control
Optional features let you decide what gets removed and what stays. Updates, browser choice, post-install configuration — it's your machine.

### 📦 Post-Install Scripts
Fine-tune Windows behaviour any time after install without ever needing to reinstall. Services, updates, and more — all scriptable, all yours.

### 🌐 Open Shell
A proper Start menu replacement that actually feels natural to use.

## 🚀 Installation

1. Download and install [AME Wizard](https://ameliorated.io)
2. Grab the latest `.apbx` release from the [Releases](../../releases) page
3. Open AME Wizard and load the playbook file
4. Read each option carefully and proceed through the steps
5. Let it run to completion and reboot when prompted

> ⚠️ This playbook makes real, permanent changes to Windows. Back up your data. Read before you run.

Once finished, check the included `scripts` folder to fine-tune your setup post-install.

## ❓ FAQ

**Is this free?**
Yes. Always. Don't pay anyone for this — if someone is charging for it they're violating our license.

**Will this work on my hardware?**
It's been tested across a range of setups but results can vary. Open an issue or ask in the Discord if something isn't working.

**Can I go back to stock Windows?**
Some changes are reversible, many are not. A clean Windows reinstall is always an option if needed.

**Does this affect my Windows license key?**
No. Your activation is untouched.

**Will my antivirus flag this?**
Possibly. The scripts operate at a low system level. Add exceptions if needed.

**Who is this for?**
Gamers, power users, and anyone tired of Windows making decisions for them. This isn't a beginner tool — it makes permanent changes. Know what you're running.
